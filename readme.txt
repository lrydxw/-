项目名称：白菜价折扣
小组成员及负责内容：
	何森鹏 :首页内容及特效，部分JSON数据书写
	刘威     :值得逛内容及详情页内容及特效，部分JSON数据书写
	闻卓	 :搜索页及列表内容及特效，部分JSON数据书写
	李锐洁 :个人中心页面及个页面之间的跳转链接
实现功能:用户注册登录，购买商品，购物车，购物车结算至订单页，搜索页和值得逛页面跳转至列表页，列表页跳转至详情页
		不登录账号无法购买商品及查看购物车和订单，退出当前账号


说明:
1、将文件解压到一个空文件夹
2、使用HBuilder或者在当前文件夹下开服务器打开 index.hmtl文件
3、数据为手写JSON 为lists.json
4、首页下部商品图点击会跳转至详情页 详情页的第一张图片 商品名 价格 折扣价会根据商品变化而变化
	详情页下部图片不会变化(这部分没写JSON数据 是固定的)，点击加入购物车会加入购物车(没有
	登录的话会跳转至登录界面)
5、点击下部首页、搜索、值得逛、个人中心会跳转至相应内容
6、由搜索页和值得逛页面点击会进入对应列表页 点击列表页内容会进入详情页  列表页左下角按钮可以更换
	列表视图，并且点击每个页面左上角返回按钮可以返回上级页面
7、由个人中心的购物车、订单可以进入对应页面(没有登录的话也会跳转登录界面)
8、购物车可以编辑，结算后的商品会可以进入订单页面查看，订单页面可以实现删除订单功能
9、有登录页面可以进入注册页面，注册成功就可以登录
9、如果需要编辑项目 需要配置 node和webpack环境 在项目下打开命令行 输入cnpm install
	或者npm install，安装成功后输入webpack -w就可以编辑项目
